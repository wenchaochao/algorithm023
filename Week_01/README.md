7.直方图的最大面积
暴力1，每个数的左右边界（比它小的左右），那么中间的就是可求的面积.  O(n^3)
for i->0, n-2
   for j-> i+1, n-1
      (i,j) -> 最小高度，area
         update maxArea
暴力2，
for i -> 0, n-1
  找到 left bound，right bound
  area = height[i] * (right - left)
  update maxArea
